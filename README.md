# 👨‍💻 Yakup Efe Çelebi - Kişisel Portfolyo

Süleyman Demirel Üniversitesi Bilgisayar Mühendisliği 3. sınıf öğrencisi olarak geliştirdiğim, yeteneklerimi, projelerimi ve iletişim bilgilerimi sergilediğim kişisel portfolyo web sitem. 



🚀 **Canlı Demo:**  [Siteyi Görüntüle](https://yec-portfolio.vercel.app)

## ✨ Öne Çıkan Özellikler
* **3D Etkileşimli Deneyim:** Sayfa kaydırıldıkça (scroll) açılan ve projeleri gösteren 3 boyutlu CSS laptop simülasyonu.
* **Özel İmleç (Custom Cursor):** Butonlarda ve linklerde manyetik hover efektlerine sahip, akıcı mouse deneyimi.
* **Canvas Animasyonu:** Hero bölümünde kullanıcı etkileşimli, arka plan parçacık (particle) animasyonu.
* **Responsive Tasarım:** Mobil, tablet ve masaüstü cihazlarla %100 uyumlu.
* **Performans Odaklı:** Sıfır dış kütüphane bağımlılığı (No frameworks). Saf HTML, CSS ve JavaScript ile maksimum yükleme hızı.

## 🛠 Kullanılan Teknolojiler
* HTML5
* CSS3 (Modern CSS Değişkenleri, Grid/Flexbox, 3D Transform)
* Vanilla JavaScript (ES6+ Modüler Yapı)
* Deployment: Vercel
